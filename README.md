## Environment variables

| Variable             | Description                                                  |
| -------------------- | ------------------------------------------------------------ |
| `PUBLIC_ENVIRONMENT` | Can be either `production` or `test` (default: `production`) |
| `PUBLIC_SPOT_URL`    | Overwrites the Spot URL (optional)                           |
