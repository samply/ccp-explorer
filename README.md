## Environment variables

| Variable                | Description                                                                                    |
| ----------------------- | ---------------------------------------------------------------------------------------------- |
| `PUBLIC_ENVIRONMENT`    | Can be either `production` or `staging` (default: `production`)                                |
| `PUBLIC_BACKEND_URL`    | Overwrites the Spot URL (optional)                                                             |
